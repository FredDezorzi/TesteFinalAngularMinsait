Bem vindo ao Fredies Bank's

Este projeto foi desenvolvido dentro de um treinamento fornecido pela Mindsait com o objetivo de identificar novos talentos na área de programação em Java e Angular.

Foram utilizadas tecnologias modernas e eficientes tanto no back-end quanto no front-end. Na camada de back-end, foi utilizado o framework Spring Boot com Java, que permite criar serviços web de maneira ágil e eficiente, além de fornecer uma arquitetura de microserviços escalável e modular. Na camada de front-end, foi utilizado o framework Angular, que é moderno e poderoso para desenvolvimento de aplicações web e mobile, permitindo a criação de interfaces ricas e interativas com o usuário, juntamente com o framework Bootstrap para garantir a consistência visual e a responsividade da aplicação em diferentes dispositivos e tamanhos de tela.

A aplicação resultante é robusta, escalável e moderna, proporcionando uma experiência de uso fluida e agradável aos usuários.
